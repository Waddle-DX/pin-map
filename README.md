# pin-map

画像の上にピンを立てて、ムーミンたちの出現箇所をリアルタイムで共有するシンプルなサイトです。選んだキャラ色のピンをクリックで配置でき、一定時間で自動的に消えます。

## 使い方
- `index.html` を開いて表示します（GitHub Pages 配信に対応）。
- 画像は `map.jpg` を用意してください。
- Firebase Realtime Database を使う場合は、`index.html` の `window.FIREBASE_CONFIG` を自分のプロジェクト設定に置き換えてください。

## 注意
- Firebase を使う場合は、公開サイト向けのルール設定が必須です。
- 更新が反映されないときはブラウザのキャッシュをクリアしてください。
